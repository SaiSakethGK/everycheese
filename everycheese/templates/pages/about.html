About CheeseAtlas

A production-grade Django application demonstrating modern full-stack engineering — REST APIs, authenticated CRUD, and a community rating system.

🏗️
Architecture
  • Django 3.1 (CBVs + mixins)
  • PostgreSQL with ATOMIC_REQUESTS
  • django-allauth authentication
  • WhiteNoise static file serving
  • 12-factor app configuration
🔌
REST API
  • Django REST Framework 3.14
  • OpenAPI 3 via drf-spectacular
  • Swagger UI + ReDoc
  • django-filter for query params
  • Session + Basic auth
🧪
Quality & DevOps
  • pytest + factory_boy fixtures
  • Coverage reporting
  • GitHub Actions CI/CD
  • Docker multi-stage build
  • pre-commit hooks

Engineering Highlights

⚡
DB-Level Aggregation

Cheese ratings use Avg() at the ORM layer, not Python loops. Queries are annotated once and reused across list and detail views.

🔒
Creator-Only Permissions

CreatorRequiredMixin combines LoginRequiredMixin and UserPassesTestMixin to enforce ownership at the view layer — no data leaks.

🔍
N+1 Free Queries

List views use select_related for creator and annotate for average score — a single query regardless of result count.
